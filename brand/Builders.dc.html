Quickstart

Four commands to localhost:3000.

# 1 · configure
cp .env.example .env

# 2 · install (pnpm workspaces)
pnpm install

# 3 · database (Prisma / SQLite)
pnpm --filter @compation/agent exec prisma migrate deploy

# 4 · run → http://localhost:3000
pnpm --filter @compation/web dev
Executor modes

Safe by default. Real on request.

EXECUTOR=fakeDEFAULT

The safe dev default — full UI, no on-chain writes.

EXECUTOR=sdkTESTNET

Real signed testnet trades — the headline path, real tx hashes.

ALLOW_MAINNET_WRITESHARD-GATED

Mainnet writes exist, but are off unless explicitly enabled.
