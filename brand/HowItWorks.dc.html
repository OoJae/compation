Architecture

Azure brains. Injective hands. A wallet that pays its own way.

Keys never touch the model. The language model decides intent; a deterministic engine decides size; Injective signs and broadcasts; and the wallet pays its own way — then earns.

Components

Six parts, each demoable on its own.

ORCHESTRATOR

Azure OpenAI via the Vercel AI SDK. Streams reasoning, calls tools, narrates — never sizes.

RISK ENGINE

Pure, deterministic hedge math. 67 unit tests. Throws typed errors, never silent failures.

INJECTIVE EXECUTOR

@injectivelabs/sdk-ts for signing; the MCP server (37 tools) for discovery. Returns real tx hashes.

X402 PAYER

Handles the 402 → sign USDC → 200 flow, and records the receipt: amount, tx, latency.

ERC-8004 IDENTITY

Identity NFT #49 in Injective's registry; the wallet set as fee recipient to earn rebates.

DECISION-TRAIL LOGGER

Persists every observation, decision, tool call, and result — rendered as a transparent timeline.

The trust split

The model reasons. The engine sizes.

This is the whole trust model. Language is good at intent and bad at arithmetic — so the LLM never touches a position size. Every number is computed by deterministic code, constraint-checked, and logged.
